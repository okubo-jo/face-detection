# 顔検出プログラム
PythonとOpenCVを使ったリアルタイムの顔検出プログラムです。

PCのカメラを使用して顔を検出し、顔の周囲に枠を表示します。また、画面左上に検出された顔の数を表示します。

## 概要
このプログラムはOpenCVのHaarCascade分類器を使って、顔検出を行います。 

カメラ映像をリアルタイムで処理し、検出された顔を四角形で囲みます。

## 機能
- PCカメラを使ったリアルタイム顔検出  
- 検出された顔を四角形で表示  
- 検出された顔の数を画面に表示  
- 「qキー」でアプリ終了  

## 使用技術
- Python
- OpenCV(cv2)
